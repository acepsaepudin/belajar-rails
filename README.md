## Deskripsi ##
`Blog` yang dibuat menggunakan framework Ruby on Rails 4. Proyek ini bersifat *open source* jadi siapa pun bisa berkontribusi, mengambil dan meyebarluaskan *source code*.

## Features ##
1. User
 	- [ ] Registrasi
 	- [ ] Login
 	- [ ] Logout
 	- [ ] Forgot Password
2. Article
	- [ ] Create Article
	- [ ] List Article
	- [ ] Article Details
	- [ ] Article Pagination
	- [ ] Article Filter
3. Management Access
	- [ ] Menu Superadmin
